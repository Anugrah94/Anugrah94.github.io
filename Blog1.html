Hacktive8 Batch 18 RedFox Week 1

Nama saya Anugrah Rezki Fauzi Nasution, biasa dipanggil Anugrah atau pendeknya Anug (with “g” not “k”). Ok, singkat cerita, saya adalah seorang fresh graduate lulusan fakultas Ilmu Komputer Universitas Sriwijaya Kota Palembang Provinsi Sumatera Selatan yang saat ini sedang menempuh pendidikan IT lanjutan di Hacktive8 Indonesia. Alasan pribadi saya bergabung di Hacktive8 adalah untuk lebih mendalami lagi dunia programming khususnya di bidang Fullstack Development dan juga karena saya mempunyai cita-cita untuk dapat bekerja di perusahaan raksasa seperti Google, Microsoft dan Apple. Saya tergabung di Hacktive8 sejak tanggal 19 dan masuk dalam rombongan batch 18 RedFox. Setelah satu minggu saya belajar di Hacktive8 saya benar-benar merasakan manfaat yang menurut saya sangat signifikan dalam mempelajari programming. Pada minggu pertama ini kami diajarkan basic tentang membangun web sederhana yang hanya baru menggunakan HTML dan CSS karena untuk bagian Javascript itu akan kami pelajari di pertemuan minggu kedua. Sebelumnya saya sudah pernah belajar mengenai HTML dan CSS saat di perkuliahan, namun ketika mengulang pembelajaran di Hacktive8 saya merasakan sebuah perbedaan, yaitu sudut pandang dalam cara mengembangkan web menggunakan HTML dan CSS. Pada saat perkuliahan saya hanya terfokus pada penerapannya secara langsung alias langsung koding. Di Hacktive8 saya “disadarkan” dan diberikan arahan agar lebih efisien dan efektif dalam menggunakan HTML CSS, arahan yang diberikan sangat bermanfaat menurut saya, karena setelah itu saya yang awalnya bisa dikatakan sering pusing dan kebingungan ketika membangun layout pada web tidak lagi mengalaminya. Walau ilmu yang diberikan masih merupakan tahap persiapan, saya sangat merasa ter-develope. Selain diajarkan mengenai HTML dan CSS kami juga dikenalkan dengan GitHub serta diajarkan cara penggunaan terminal Git. Ini adalah hal cak sangat baru bagi saya, karena sebelumnya saya tidak pernah menggunakan platform GitHub. Pada materi ini kami diberitahu bahwa dalam mendevelope sebuah program sekarang tidak serumit dahulu. Menurut Wikipedia, GitHub adalah layanan penginangan web bersama untuk proyek pengembangan perangkat lunak yang menggunakan sistem pengontrol versi git dan llayanan hosting internet. Hal ini banyak digunakan untuk kode komputer. Platform ini memberikan kontrol akses dan beberapa fitur kolaborasi seperti pelacakan bug, permintaan fitur, manajemen tugas, dan wikiuntuk setiap proyek. Singkatnya GitHub ini bisa kita katakan sebagai cloud dan medsosnyanya para programmer. Disini kita bisa mengapload atau bahasa Gitnya push file ke folder online kita yang disebut repository dan ini bersifat public (open source), yang artinya siapa saja yang menggunakan GitHub dapat mengambil dan mengakses file yang kita upload. Sebelum dapat memanfaatkan GitHub, kita diharuskan paham cara dalam penggunaan Git. Menurut Wikipedia Git adalah perangkat lunak pengontrol versi atau proyek manajemen kode perangkat lunak yang diciptakan oleh Linus Torvalds, yang pada awalnya ditujukan untuk pengembangan kernel Linux. Dari Git inilah kami diajarkan cara-cara mengontrol komputer tanpan menggunakan UI (user interface), penggunaan terminal semacam ini biasanya digunakan dalam Linux dan juga dalam menggontrol sebuah server. Dalam pembelajaran ini Git digunakan sebagai media kontrol untuk mensinergikan antara perangkat komputer pribadi dan GitHub. Pada platform GitHub, kami juga diajarkan cara memanfaatkan layanan free hosting yang ditawarkan oleh GitHub agar web sederhana yang telah kami kembangkan dapat diakses secara online. HTML, CSS, Terminal, Git dan GitHub inilah yang kami pelajari dalam pertemuan-pertemuan kami diminggu pertama. Selain materi-materinya lingkungan yang ditawarkan oleh Hacktive8 sangat mendukung proses belajar. Mulai dari siswa-siswanya, pada instrukturnya dan fasilitas yang disediakan. Dalam waktu 1 minggu saya pribadi sudah sangat mengerti mengenai materi yang diajarkan dan merasa sangat nyaman dengan kondisi lingkungan Hacktive8. Dari pengalaman minggu pertama disini, daya sangat puas dan yakin kedepannya saya bisa terus berkembang di Hacktive8 dan menjadi Fullstack Developement yang profesional sertadapat mencapai cita-cita saya.
